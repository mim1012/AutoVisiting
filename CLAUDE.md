## 🔧 기술 스택

### 현재 사용 중
- **언어**: Python 3.8+
- **브라우저 자동화**: Selenium + Undetected ChromeDriver
- **프론트엔드**: Tkinter (GUI), Flask (웹)
- **설정 관리**: YAML, JSON
- **로깅**: Python logging + colorlog

### 도입 예정
- **데이터베이스**: SQLite → PostgreSQL
- **캐싱**: Redis
- **메시지 큐**: RabbitMQ
- **모니터링**: Grafana + Prometheus
- **CI/CD**: GitHub Actions
